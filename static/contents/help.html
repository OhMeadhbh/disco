The Discovery Web Desktop

Hi! Thanks for trying out the Discovery Web Desktop (we call it "Disco" for short.) This document will hopefully get you on your way to using Disco effectively.

What is Disco?

Disco is a web application that emulates a desktop experience. You do the same things with Disco you would do with your Macintosh, Windows or Gnome desktop interfaces. But with Disco, you're doing it in the cloud. Changes to documents and applications (even the Disco desktop) are saved on a remote server. Changes you make on one computer will automagically show up on any other computer you log in from. A web-based desktop also has the benefit of centrally managed backups and easy software distribution.

Disco was designed to be used on both desktop and tablet computers. But unfortunately it's only currently compatible with Mozilla, Chrome and Safari web browsers.

Okay. How do I do things with Disco?

Working with Disco is very much like working with any other desktop interface. Mouse users point at screen controls with the mouse and activate them by clicking. Tablet users just touch the interface element they're interested in.

Along the top of the screen is the Menu Bar; clicking menu items on the menu bar will either cause a sub-menu to open or an action to be performed. Portions of the Menu Bar will change as you move between applications, though all apps share the glider menu, the help window and the full screen control, denoted with these icons:

Glider Menu
Help Window
Full-Screen Control

The Glider Menu contains a number of sub-menu items which give you access to application settings and commonly used accessories such as a calculator and a clock. The Help Window icon opens this help window. The Full Screen Control toggles in and out of full screen on browsers that support it.
